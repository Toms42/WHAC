# WHAC

## Components
1. LIDAR System (rplidar + our lidar code)
2. ZED Mini + ORB-SLAM2 (orb-slam2 + our slam code)
3. Motion Planning
4. Robot

## Code
In total there will be 6 ROS packages. Two for LIDAR, two for SLAM, one for motion planning and one for robot control.
